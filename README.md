# AI-Appointment-Scheduling-and-Reminder-Agent

An AI-powered appointment booking and reminder system built using **n8n**, **Groq LLM**, **WhatsApp Cloud API**, **Google Calendar**, **Google Sheets**, and **Gmail**.

The AI agent understands natural language, checks calendar availability, books appointments automatically, stores patient information, and sends confirmation and reminder messages.

---

## Features

- AI-powered appointment booking
- Natural language conversation
- Google Calendar integration
- Calendar availability checking
- WhatsApp confirmations
- Gmail confirmations
- Automated reminder workflow
- Google Sheets database
- Memory-enabled AI Agent
- Multi-tool orchestration using n8n

---

## Tech Stack

- n8n
- Python
- Groq API
- WhatsApp Cloud API
- Google Calendar API
- Gmail API
- Google Sheets API
- JavaScript
- Webhooks

---

## Workflow

User → WhatsApp → AI Agent → Check Calendar → Create Event → Save Appointment → Gmail Confirmation → WhatsApp Confirmation → Scheduled Reminder
